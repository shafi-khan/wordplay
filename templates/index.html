WordPlay

Following is the list of endpoints available:

/random - Retrieves a random word

/words/{insert_your_word}/frequency - Returns the frequency score of a given word

/words/{insert_your_word}/syllables - Returns number of syllables in a given word

/words/{insert_your_word}/pronunciation - Returns pronunciation information of a given word

/words/{insert_your_word}/rhymes - Returns rhymes of a given word

/words/{insert_your_word}/examples - Returns example uses

/words/{insert_your_word}/definitions - Returns definition(s) of a given word

/words/{insert_your_word}/synonyms - Returns synonyms

/words/{insert_your_word}/antonyms - Returns antonyms




Please note: You can replace {insert_your_word} with a word of your choice. Defaults to the word python
